# Project Readme

## О проекте

Этот проект представляет собой интерфейс для работы с пользователями: создание, редактирование, удаление пользователей и авторизация.

## Возможности

* Авторизация пользователей
* Просмотр списка пользователей
* Создание нового пользователя
* Редактирование существующего пользователя
* Удаление пользователя

## В будущем можно улучшить проект:

1. **Вынести константы в отдельный файл**

   * Например, URL API, ключи localStorage, цвета и тексты кнопок.
   * **Причина:** централизованное управление значениями, проще менять их без правки кода в нескольких местах.

2. **Использовать стейт-менеджер**

   * Например, Zustand, Redux или MobX.
   * **Причина:** упростить работу с глобальным состоянием, избежать передачи props через несколько уровней компонентов, улучшить поддержку авторизации и состояния модалок.

3. **Добавить типизацию и интерфейсы для всех API-запросов**

   * Использовать TypeScript для всех функций, чтобы минимизировать ошибки и улучшить автокомплит.

4. **Улучшить UX модалок**

   * Добавить централизованный спиннер или глобальный loader.
   * Обрабатывать ошибки API централизованно через уведомления.
